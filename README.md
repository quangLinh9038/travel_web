## Installation

Locally installation

 - Backend app: 
   Make sure that you have node.js and npm package installed
   ```
   cd server.js
   ```
   ```
   npm install 
   ```
   ```
   npm run dev
   ```
- Frontend app: 
  ```
  cd front/src/App.js
  ```
  
  ```
  npm install
  ```
  
  ```
  npm start
  ```
